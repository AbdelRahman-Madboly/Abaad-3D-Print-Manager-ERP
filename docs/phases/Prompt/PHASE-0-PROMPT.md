## Why this phase exists

Previous phase reports claimed work was done that doesn't appear to be on
disk. Before writing a single line of code, we need ground truth: what files
exist, what tests actually pass, what is hardcoded that shouldn't be, and
what the DB schema really looks like. This report becomes the input to every
subsequent phase prompt.

### Task 8 — DB schema

```bash
python3 -c "
import sqlite3, os
db_path = 'data/abaad_v5.db'
if not os.path.exists(db_path):
    print('DB does not exist yet')
else:
    conn = sqlite3.connect(db_path)
    conn.row_factory = sqlite3.Row
    tables = conn.execute(\"SELECT name FROM sqlite_master WHERE type='table' ORDER BY name\").fetchall()
    for t in tables:
        name = t['name']
        cols = conn.execute(f'PRAGMA table_info({name})').fetchall()
        print(f'\\n{name}:')
        for c in cols:
            print(f'  {c[\"name\"]} {c[\"type\"]} {\"NOT NULL\" if c[\"notnull\"] else \"\"} {\"DEFAULT \"+str(c[\"dflt_value\"]) if c[\"dflt_value\"] is not None else \"\"}')
    conn.close()
"
```

Report the full schema. If the DB doesn't exist yet, note that and report
what `DatabaseManager._create_tables()` would create instead.
