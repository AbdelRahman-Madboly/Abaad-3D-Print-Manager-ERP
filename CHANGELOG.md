## [5.0.0] — 2026-06-16

### Added
- Unified Dashboard tab with Action Center, Headline Numbers, Charts,
  Printer Utilization, and Detailed Breakdowns sections
- First-run setup wizard (4 steps: company info, filament, printer, cost defaults)
- Per-printer print job tracking on order status transition to "Ready"
- `get_orders_needing_attention()` — overdue Ready orders + unpaid delivered orders
- `get_printers_needing_maintenance()` — nozzle wear alerts
- Tenant brand system — company identity configurable via `settings` SQLite table
- App header shows company name and subtitle from settings
- Login screen shows company name and logo from settings
- `PDF_FOOTER_CREDIT` constant — `"Generated by Abaad ERP"` on every PDF
- `get_print_items_totals()` — single aggregate query for total weight/time printed
- GitHub Actions CI (lint with ruff + test with pytest on push/PR to develop/main)

### Fixed
- `auth_manager.py` no longer prints `admin / admin123` to stdout (now `log.info`)
- Settings tab DB access (was calling nonexistent `execute_query`)
- PDF service company info loading (was calling nonexistent `get_settings_by_keys`)
- Analytics aggregation methods (monthly revenue, order status, filament usage)
- Status bar filament key (`available_weight_g` not `total_available`)
- `total_weight_printed` / `total_time_printed` always returning 0 (fixed via
  `get_print_items_totals()` aggregate, not per-order property sum)
- Setup wizard now fires on first launch (was never wired into boot sequence)
- `config.DEFAULT_SETTINGS` missing required keys (caused crash on first run)
- `COMPANY` dict neutralized — no hardcoded business contact info in source
- `pyproject.toml` build backend corrected (`setuptools.build_meta`)

### Removed
- v4 migration script and tests
- Hardcoded Abaad business contact info from `config.COMPANY` (phone, address, social)
- Separate Stats and Analytics tabs (merged into Dashboard)
- Dead `analytics_tab.py` (superseded by `dashboard_tab.py`)
- `requirements.txt` (all deps now declared in `pyproject.toml`)
